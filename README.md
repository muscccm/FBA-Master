## 这是什么？

FBA Master 是一款专为亚马逊卖家打造的利润计算工具。无论你是刚入行的新手还是经验丰富的老卖家，它都能帮你快速算清每一件产品从采购到销售的全部费用，让你在定价时心中有数。

软件支持美国、德国、日本三大亚马逊站点，内置最新的 FBA 配送费用和佣金费率，输入产品信息后一键计算，所有费用明细一目了然。

---



## 主要功能

### 🌍 三大站点，一键切换

支持亚马逊美国站（USD）、德国站（EUR）和日本站（JPY）。切换站点后，产品类目、物流方式、VAT 税率、默认货币等全部自动更新，无需手动调整。

### 📊 三种计算模式，灵活应对不同场景

- **模式一 · 利润率计算**：输入采购成本和销售价格，自动算出利润率。适合已经确定售价、想看利润空间的场景。
- **模式二 · 建议售价计算**：输入采购成本和目标利润率，自动推算出应该定多少价。适合新品定价参考。
- **模式三 · 成本控制计算**：输入目标利润率和销售价格，反推出最高能接受的采购成本。适合在已知售价的情况下筛选供应商。

### 📦 完整的产品信息录入

- 产品名称、长宽高尺寸、重量
- 尺寸单位支持厘米（cm）和英寸（in）自由切换，切换时数值自动换算
- 重量单位支持千克（KG）、盎司（OZ）和磅（POUND）自由切换
- 产品类目选择（各站点类目不同，自动匹配）
- 美国站额外支持选择产品类型（普通商品、服装、危险品）
- 物流方式选择（各站点物流方式不同，自动匹配）

### 💰 全面的费用计算

软件会自动计算以下所有费用，每一项都支持多种货币显示：

- **FBA 配送费用**：根据产品尺寸、重量和站点规则自动计算
- **佣金**：根据产品类目和售价自动计算佣金费率和佣金金额
- **头程运费**：根据物流方式单价和计费重量（取实际重量和体积重量中较大值）计算
- **推广费用**：按你设定的推广费率计算
- **退货费用**：按你设定的退货费率计算
- **VAT 税费**：按站点默认或自定义税率计算
- **利润额**：扣除所有费用后的实际利润

### 📈 关键参考数据

除了基本的利润计算，软件还会自动算出几个非常实用的参考数据：

- **平本售价**：利润率刚好为 0% 时的最低售价，低于这个价就亏钱了
- **20% 利润售价**：想要 20% 利润率时应该定的价格
- **20% 利润成本控制**：在 20% 利润售价下，采购成本最多不能超过多少
- **尺寸换算**：自动显示厘米和英寸的对照结果
- **重量换算**：自动显示不同重量单位的对照结果
- **尺寸分段**：显示当前产品在亚马逊的尺寸分段归类（影响 FBA 费用）

### 🚚 物流单价自定义

每个站点都有不同的物流方式，你可以自由设置每种物流的单价：

- 美国站：红单、空运、海运、铁运/以星、卡航/美森
- 德国站：空运、海运、铁运/以星、卡航/美森
- 日本站：空运、海运

物流费用货币支持 CNY、USD、GBP、JPY、EUR 五种选择。

### 💹 实时汇率

- 启动时自动获取最新汇率（USD、GBP、JPY、EUR 对 CNY）
- 工具栏实时显示当前汇率和更新时间
- 支持手动刷新汇率
- 网络不可用时自动使用内置默认汇率，不影响正常使用
- 所有费用字段都支持在五种货币之间自由切换显示

### 📊 投入产出分析

帮你评估一个产品值不值得做：

- 输入模具费用和预计销量
- 自动计算前期总投入（采购成本 + 头程运费 + 模具费用）
- 自动计算模具回本销量和整体回本销量
- 自动计算预计销售额、预计利润额
- 自动计算投入产出比（ROI）

### 📋 产品对比表格

这是选品时最实用的功能之一：

- 计算完一个产品后，点击"添加到对比"，产品数据就会保存到对比表格中
- 可以添加多个产品，横向对比所有费用和利润数据
- 支持统一切换对比表格的尺寸单位、重量单位和货币
- 双击某一行可以把该产品数据重新加载回输入区，方便修改后重新计算
- 支持删除选中的对比项
- 支持导出为 Excel 文件（.xlsx），方便分享或存档
- 支持从 Excel 文件导入对比数据


### ⚙️ 灵活的配置管理

所有计算规则都以 JSON 配置文件的形式存储，你可以：

- 在设置界面中直接编辑 FBA 费用规则、佣金费率、站点配置、API 设置
- 亚马逊每年调整费率时，只需修改配置即可，无需等待软件更新
- 支持一键导出所有配置为 JSON 文件（自动生成带时间戳的文件名）
- 支持从 JSON 文件导入配置，方便在不同电脑间迁移设置
- 导入时自动验证文件格式，格式错误会提示而不会覆盖现有配置

### 🎨 主题色自定义

- 支持日间模式和夜间模式切换
- 内置 6 套精心设计的配色方案：默认紫罗兰、海洋蓝、森林绿、暖阳橙、玫瑰粉、石墨灰
- 每种颜色都可以单独自定义，修改后实时预览效果
- 自定义配色会自动保存，下次启动自动恢复
- 配色方案包含在配置导入/导出中，换电脑也能带走你的个性化设置

### 🖥️ 舒适的使用体验

- 现代化卡片式界面设计，信息层次清晰
- 大字体醒目显示计算结果
- 可编辑字段和只读结果字段有明显的视觉区分
- 窗口大小自动适配屏幕分辨率，小屏幕也能正常使用
- 支持滚动查看完整内容
- 所有标准对话框（颜色选择器、文件选择器等）已汉化

## 使用说明

### 快速开始

1. 启动软件，默认为美国站、模式一（利润率计算）
2. 在左侧填写产品信息：名称、尺寸、重量、类目、物流方式
3. 在右侧填写成本信息：采购成本、采购运费、推广费率、退货费率
4. 填写销售价格
5. 点击"★ 计算"按钮，所有费用和利润数据自动计算完成

### 切换站点

点击设置按钮，在设置界面中选择站点（美国/德国/日本）。切换后类目、物流方式、税率等自动更新。

### 切换计算模式

点击顶部工具栏的"模式一""模式二""模式三"按钮切换。当前选中的模式会以橙色高亮显示。

### 产品对比

1. 计算完一个产品后，在底部对比表格区域点击"+ 添加到对比"
2. 修改产品信息，重新计算，再次添加
3. 重复以上步骤，添加所有想对比的产品
4. 在对比表格中横向比较各产品的费用和利润

### 导出配置

设置界面底部点击"导出配置"，文件名会自动生成（如 `FBA Master_EXPORT_2602211556.json`），选择保存位置即可。

### 导入配置

设置界面底部点击"导入配置"，选择之前导出的 JSON 文件，所有配置（包括 FBA 费率、佣金规则、主题颜色等）会一次性导入。
